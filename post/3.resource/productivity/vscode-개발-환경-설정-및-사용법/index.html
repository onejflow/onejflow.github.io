vscode 기본 설정

  • 기본설정
    • settings 에서 검색
      • bracket pair = True or active true는 모든 가이드라인, active는 현재 커서 기준
      • white space -> boundary (single space제외 표기)
      • cmd k + cmd t -> 테마 선택 키보드 방향으로 미리보기후 맘에드는거 선택
      • work wrap 자동 줄바꿈 editor: work wrap -> on 해주기
      • cursor blinking
      • explorer.compactFolders 검색 후 Explorer: Compact Folders 체크 해제
    • 마우스 휠로 글자크기 조정
      • preference > settings > text editor > mouse wheel zoom > 체크
  • 파일 자동 저장
    • preference > settings > text editor > file > autosave -> afterDelay 로 설정하여 자동 저장
  • 파일 자동 포맷팅
    • preference > settings > text editor > formatting > format on save > 체크
  • 괄호 색 설정
    • bracketPairColorization menabled 검색 Editor > Braket Pair Colorization JEnabled, Editor > Guides:Bracket pair
  • 디렉토리에 __pycache__ 같은 보여질 필요 없는것들 제외하기
    • preference > settings > files:exclude 검색 > add pattern 에 추가하기 **__pycache__
  • 디렉토리 폴더 구조 겹침 없애기
    • preference > settings > explorer.compactFolders 검색 > Explorer: Compact Folders 체크해제
  • 한 화면에 긴 코드 보기 (word wrap)
    • settings > wordwrap 검색 > wordwrap on
  • 해당 셋팅 및 익스텐션들을 연동하기 위해 settings sync on 하기

VSCODE Extension

  • 파이썬 플젝을 위한 python, pylance, jupyter
  • Trailing Spaces -> 스페이스 표시로 불필요한 스페이스 확인 가능
  • ruff: isort에서 해주는 임포트 순서 체크, black 파이썬 포매터를 대신하여 사용
  • mypy: type 체크를 위한 도구 (변수와 함수 인자에 타입 어노테이션이 되어 있어야 의미 있는 테스트 가능)
  • autoDocstring - Python 독스트링 생성
  • meterial icon theme: 아이콘
  • indent rainbow: 들여쓰기
  • TodoTree: Todo 주석을 남길시 볼 수 있음
  • file-tree-generator: 이건 뭐 필요하면하고 아님말고
  • csv, json같은거 미리보기 Excel Viewer / Data Preview 두개 좋음 특히 data preview는 그래프도 그릴수 있음 이건 알아두기!
  • Better Dockerfile Syntax
  • Even Better TOML
  • YAML
  • Shell-format
  • project manager: 프로젝트 워크스페이스 만들어서 해당폴더 갈수있게 해줌
  • code runner
  • inline sql: SQL구문 하이라이트
  • DBCode: db 연결해서 볼 수 있음
  • Marp for VS Code: 마크다운 기반 ppt 만들기
  • vscode-pdf
